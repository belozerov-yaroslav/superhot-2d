Основной файл игры - superhot-dev.py.
Управление по умолчанию:
E - выстрел,
Стрелочки - поворот в разные направления,
Пробел - сделать ход туда, куда смотрит персонаж.
Чтобы поменять управление, надо отредактировать файл - config.py.
Правила:
Как игрок, так и враги стреляют по прямой линии, уничтожая первый объект в который попадёт лазер.
Враги двигаются или стреляют только тогда, когда двигается или стреляет игрок, поворот не считаеться за ход.
Также на карте есть бочки с топливом, через них нельзя пройти, и при попадании лазера,
они взрываются и уничтажают все в радиусе одной клетки (по-диагонали тоже).
На карте есть коробки, через которые нельзя пройти, и можно просто уничтожить.
Если игрок стреляет, то его лазер нагревается, и если игрок переместится в другую клетку,
то его лазер полностью охладится.
Лазер может выдержать 3 выстрела без охлаждения.
Внешний вид:
Игрок - робот в виде двух голубых треугольников.
Противники - шарообразные красные роботы.
Бочка с топливом - красный круг со значком огнеопсаности.
Коробка - зелёный квадрат.
